# Vue Demo

## Project description
1. *vue + vue-router + vuex*
2. *element-plus*
3. *node version ^12.22.0 || ^14.17.0 || >=16.0.0*
4. *webpack 4+*


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
